## 🎨 主要变更

### Logo 设计风格
从**几何线条极简风格** → **中国书法泼墨风格**

| 对比项 | v1.0（旧版） | v2.0（新版） |

| 笔画粗细 | 2-3px 细线 | 6-10px 粗壮笔画 |
| 设计风格 | 几何极简 | 书法泼墨写意 |
| 装饰元素 | 虚线 | 飞白、墨点、印章 |
| 科技感 | 中性 | 蓝色渐变强化 |
| 文化属性 | 国际化 | 东方中国风 |
| 视觉冲击 | 低调克制 | 有力量感 |

## ✨ 新版设计特点

### 1. 粗壮书法笔画
- **左右竖画**：6-8px，墨色渐变（#0f172a → #334155）
- **中间斜杠**：8-10px，蓝色渐变（#1e40af → #60a5fa）
- 有起笔、收笔的笔锋细节

### 2. 书法艺术细节
- **飞白效果**：3-4处，模拟笔触快速运动
- **墨点装饰**：4-6个随机小点
- **笔锋细节**：起笔尖锋，收笔墨滴

### 3. 融合元素
- **蓝色渐变**：AI科技感，数据流动
- **红色印章**：右下角小圆形，中国传统文化
- **整体克制**：保持商务专业感

## 🎯 使用指南

### 在React Web中使用
```tsx
import { Logo } from '@/app/components/brand/Logo';

// 深色版（浅色背景）
<Logo variant="dark" size="lg" />

// 浅色版（深色背景）
<Logo variant="light" size="xl" />
```

### 在微信小程序中使用
1. Logo SVG源文件：`/exports/logo-dark.svg` 和 `/exports/logo-light.svg`
2. **PNG转换**：手动转换为PNG后，放入 `/exports/icons-png/` 目录
   - 推荐文件名：`logo-dark.png` 和 `logo-light.png`
   - 推荐尺寸：144×144px（@3x高清）
3. 深色版用于浅色背景，浅色版用于深色背景
4. 最小尺寸建议48px以上，保证细节清晰
5. 在小程序中使用时，将PNG复制到小程序项目的 `/images/` 目录

**转换工具**：https://svgtopng.com/ （设置144×144px）

### 尺寸建议
| 场景 | 推荐尺寸 | 说明 |
|------|---------|------|
| 启动页 | 96×96px | 主视觉焦点 |
| 页面头部 | 48×48px | 标准展示 |
| TabBar | 32×32px | 导航图标 |
| 最小尺寸 | 24×24px | 保证清晰度下限 |

## 🖼️ 设计理念

### 为什么选择书法风格？

1. **文化契合**
   - 中日商务背景，书法是东亚共通语言
   - 展现企业文化底蕴和专业性

2. **视觉差异化**
   - 区别于扁平化、极简主义泛滥
   - 独特的文化识别度

3. **传统与现代融合**
   - 书法 = 传统、权威、稳重
   - 蓝色科技 = AI、创新、未来
   - 完美平衡

4. **目标受众共鸣**
   - 中高层管理者欣赏文化内涵
   - 符合金融/咨询行业气质
